# Frontend Mentor - Digitalbank landing page solution

## Overview
This project is a static & Responsive landing page for Digital Bank, built using HTML, CSS & Bootstrap to match the design provided by [Frontend Mentor](https://www.frontendmentor.io/challenges/digital-bank-landing-page-WaUhkoDN).

### The challenge
Bellow challenge has been completd as part of this project,

- View the optimal layout for the site depending on their device's screen size

## My process

I began by structuring the page with semantic HTML tags for better accessibility. Then used the Bootstrap 5 framework to handle the responsive layout and grid system along with custom CSS to  match the design provided by Frontend Mentor.

### Customization & Implementation
Created the responsive UI by adding different Bootstrap classes in every section of the page for desktop and mobile device,

- Responsive Image in Hero section: Used Bootstrap’s display utilities like d-none d-md-block and d-md-none, to achieve the proper loading of img element.
- Overylay image : Applied the utilities, position-relative and position-absolute along with the translate-middle to center the phone mockup on top of the background image. 
- Order Utilities - Included the classes such as order-2 order-md-1, to change the column order in hero section for mobile device.
- Combining the layout: Used the Bootstrap's Grid system and CSS flex layout to achieve the responsiveness and alignment.
